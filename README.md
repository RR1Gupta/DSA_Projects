# DSA_Projects
This repository contains my Data Structures and Algorithms (DSA) practice projects implemented in C++. It is a collection of solved problems and implementations organized by topic. The purpose of this repository is to practice, learn, and demonstrate problem-solving skills using core DSA concepts.

Employee Management System
Overview
This is a simple Employee Management System written in C++. It allows the user to manage employee information, including adding new employees, displaying existing records, searching for employees, and sorting records by name and ID.
The program uses classes, vectors, and file handling to store and manipulate employee data.

Features
Add Employee – Enter employee name, ID, and related file information.
Display Employees – Show a list of all stored employees.
Search Employee – Search for employees by name or ID.
Sort Employees – Sort employee records alphabetically by name or numerically by ID.
Input Validation – Ensures valid names and IDs are entered.
